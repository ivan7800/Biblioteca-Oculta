# Arcanum 404: Biblioteca Oculta

**Versión 2.1.4**

PWA offline-first en HTML, CSS y JavaScript puro. Sin dependencias externas, sin APIs, sin tracking y lista para GitHub Pages.

## Módulos

- Sala principal con acceso a todos los libros.
- Tarot: carta única, tirada de 3, Cruz 404, historial y comparación.
- Ouija ficticia: tablero narrativo simulado, sin afirmaciones paranormales reales.
- Diario paranormal: registro separado de experiencias, lugar, intensidad, etiquetas y búsqueda.
- Runas: Elder Futhark completo, tiradas, significados e historial.
- I Ching: monedas virtuales, líneas mutables, hexagrama y archivo.
- Sueños: diario, calendario, símbolos recurrentes, estadísticas y exportación.
- Libro Lunar: calendario mes a mes, fase diaria, hitos aproximados y correspondencias.
- Grimorio: correspondencias de hierbas, piedras, metales, planetas, elementos y direcciones.
- Bestiario Prohibido: folclore mundial en clave cultural e histórica.
- Codex Symbolorum: alquimia, astrología, sigilos, árbol de la vida y simbología comparada.
- Ritual Generator: rituales ficticios para narrativa, rol y atmósfera.
- Consulta rápida: lectura combinada con carta, runa, símbolo, luna y correspondencia diaria.
- Comparador: comparación de consultas guardadas para detectar repeticiones, diferencias y evolución simbólica.
- Archivo 404: historial visible, búsqueda, borrado selectivo, backup, importación y exportación JSON/TXT.
- Cámara de Ecos: audio procedural local sin copyright con presets ambientales.

## Privacidad

Todo se guarda en `localStorage` del navegador. No hay servidor, cuentas de usuario, analítica ni envío de datos.

## Publicación en GitHub Pages

1. Sube todos los archivos de esta carpeta al repositorio.
2. Activa GitHub Pages desde `Settings > Pages`.
3. Selecciona la rama principal y la carpeta raíz.
4. Abre la URL pública del proyecto.

## Nota

Proyecto simbólico, cultural, creativo y de entretenimiento. No sustituye consejo profesional ni afirma resultados sobrenaturales.
