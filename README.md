# Arcanum 404: Biblioteca Oculta

**Versión 2.1.0 PRO**

PWA offline-first en HTML, CSS y JavaScript puro. Sin dependencias externas, sin APIs, sin tracking y lista para GitHub Pages.

## Módulos incluidos

- Sala principal con acceso a todos los libros.
- Libro del Destino: Tarot con carta única, tirada de 3, Cruz 404, historial y exportación.
- Libro de los Espíritus: Ouija ficticia/narrativa con sesiones guardadas.
- Diario paranormal: registro separado de experiencias, lugares, intensidad, etiquetas, patrones, búsqueda y exportación.
- Libro de las Runas: Elder Futhark completo, tirada de una runa, tres runas, significados e historial.
- Libro de los Hexagramas: monedas virtuales, líneas mutables, hexagrama, archivo y comparación.
- Libro de los Sueños: diario, calendario, símbolos recurrentes, estadísticas y exportación.
- Libro Lunar: calendario perpetuo mes a mes, fase diaria, próximos hitos lunares y vista anual simbólica.
- Grimorio: enciclopedia ampliada de hierbas, resinas, piedras, metales, planetas, astros, direcciones y elementos.
- Bestiario Prohibido: folclore mundial en clave cultural e histórica.
- Codex Symbolorum: alquimia, astrología, runas, sigilos, árbol de la vida y simbología comparada.
- Ritual Generator: rituales ficticios para narrativa, rol y ambientación.
- Consulta rápida: lectura combinada con tarot, runa, símbolo, grimorio y luna.
- Comparador de consultas: cruza dos lecturas del archivo y detecta símbolos comunes, diferencias y evolución.
- Archivo 404: historial visible, búsqueda, borrado selectivo, exportación JSON/TXT e importación de backup.
- Cámara de Ecos: audio procedural local sin copyright con presets.

## Privacidad

Todo se guarda en `localStorage` del navegador. No hay servidor, cuentas, cookies externas ni envío de datos.

## Publicación en GitHub Pages

1. Sube todo el contenido de la carpeta `arcanum404` o la raíz del proyecto.
2. En GitHub, entra en **Settings → Pages**.
3. Selecciona la rama `main` y la carpeta raíz.
4. Espera a que GitHub Pages publique el sitio.
